# O git

Git je verzovací systém, a je moc fajn.


# Pojmy
* Commit - změna v čase
* Push - odeslání commitů na server

# O historii

Git byl vymyzšlen Linusem Torvaldsem. Někdy v 70 letech byla napsána první verze.
